# 프로젝트 개요

## 프로젝트명
ILHOS_Project_DLab

## 목적
내부 업무용 ERP 스타일 시스템과 분석 도구를 단계적으로 구축한다.

## 추진 방향
- 로그인 가능한 내부 시스템 구축
- 권한별 메뉴 제어
- 기준정보 / 구매 / 생산 / 분석 기능 확장
- MSSQL 기반 업무 데이터 관리

## 1차 개발 범위
- 로그인
- 대시보드
- 메뉴 구조
- 사용자/권한 구조
- 거래처관리
- 품목관리

## 기술 스택
- Python
- Streamlit
- MSSQL
- SQLAlchemy
- pyodbc
- Pandas
